## Beschreibung
Die Fibonacci-Folge ist eine Zahlenreihe, bei der jede Zahl die Summe der beiden vorangegangenen ist und die normalerweise mit 0 und 1 beginnt.

Diese einfache Bibliothek dient der Berechnung von Fibonacci-Zahlen bis zu einer bestimmten Grenze. Sie ermöglicht es Ihnen, verschiedene Eigenschaften wie den höchsten Primfaktor, die Parität und mehr zu berechnen.

## Installation
```bash
go get go.mcds.moe/fibonacci
```

## Benutzung
```go
package main

import (
    "fmt"
    "go.mcds.moe/fibonacci"
)

func main() {
    // Berechnet die 5. Fibonacci-Zahl rekursiv
    fmt.Println(fibonacci.Fib(5))

    // Berechnet die 10. Fibonacci-Zahl iterativ
    fmt.Println(fibonacci.FibIt(10))

    // Berechnet die 25. Fibonacci-Zahl mit mehreren Worker-Threads
    fmt.Println(fibonacci.FibMultithreaded(25))
}
```
