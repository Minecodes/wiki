## Einrichtung
<Steps>

1. Voraussetzungen
    - Ein Bluesky-Konto für den Bot
    - bunjs auf deinem Computer installiert

2. Repository klonen
    ```bash
    git clone https://github.com/Minecodes/WttrIn-Reutlingen
    ```

3. Abhängigkeiten installieren
    ```bash
    bun install
    ```

4. Erstelle eine `.env` Datei im Hauptverzeichnis des Projekts und füge folgenden Inhalt hinzu
    ```env
    BSKY_USERNAME=benutzername.bsky.network
    BSKY_PASSWORD=passwort
    BOT_CITY=Reutlingen
    ```
    Ersetze `benutzername.bsky.network` und `passwort` mit deinen Bluesky-Zugangsdaten

5. Bot starten
    - Mit bun
        ```bash
        bun start
        ```
    
    - Mit Docker Compose
        ```bash
        docker compose up -d
        ```

</Steps>
